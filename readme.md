# Git, github и пулл реквесты
Склонируйте данный репозиторий, создайте новую ветку из мейна, исправьте ошибки, сделайте код безопасным (исправлять массив arr и файл input.txt нельзя, 
если потенциально возникнет ошибка - нужно правильно ее обработать)
и отправьте запрос на слияние (пулл реквест) в мой репозиторий. После создания пулл реквеста на его странице в github должны появиться красные кресты и сообщения о том,
что требуется review - это нормально, значит все получилось, и пулл реквест ожидает одобрения.
Скопируйте ссылку на страницу пулл реквеста в гитхабе и отправьте ее на платформу, это и будет результатом выполнения дз.
